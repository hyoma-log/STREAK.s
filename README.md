# STREAK.s
## サービス概要

**「せっかく覚えたIT知識がすぐに抜けてしまう」「本を読んでも、実務コードが書ける/読めるか不安」、**この**未経験・初級エンジニア**が直面する知識定着と実践力の悩みを、**AIコードレビューと科学的な復習サイクル**で解決するプラットフォームです。

プログラミング学習の初期段階では、インプットした**大量の新しいIT知識**が頭の中で整理されず、自信を失いがちだと思います。

STREAK.sは、あなたが学習したばかりの技術トピックを登録するだけで、A I が「超アウトプット型」のクイズを自動生成し、**忘却曲線の最適なタイミング**で出題します。

これは、あなたの**IT知識を確かな定着**へと導き、**初級から中級へのステップアップ**を支える**実務に役立つコード実装力**に繋げる、**エンジニアに特化**した学習習慣プラットフォームです。

---

## このサービスへの思い・作りたい理由

プログラミングを学び始め、知識が少ない状態だった私は痛感しました。

「新しいIT知識は増えるのに、すぐに記憶から抜け落ちてしまう。特に、それが実務で使えるレベルに定着しているのか全く分からない」という現実です。

毎日インプットしても、1週間後にはもう思い出せず、**「何を復習すべきか」「自分の真の苦手は何か」も分からず、ただ自信と不安**だけが残りました。

そんな中で出会ったのが「忘却曲線」の考え方。

“人は忘れる生き物である”なら、その**忘れるタイミングを科学的に特定**し、「忘れる前に叩き直す仕組み」を作ればいい。

そう思って生まれたのがこのSTREAK.sです。

このアプリは、「IT知識の定着に悩み、実務レベルへのステップアップに不安を感じていた」過去の自分の悩みを解決するためのツールです。

そして、同じように知識定着に悩み、初級から中級へのステップアップを目指す多くのエンジニアにも役立てるものにしたいと思っています。

## サービスの利用イメージ

1. ユーザーは、自分が学んだ内容（例：RailsのN+1問題など）を登録。
2. AIが登録内容をもとに、**クイズ形式の復習問題**を自動生成。
3. 忘却曲線に基づいて、**「今復習すべき問題」**が毎日出題される。
4. 正答・誤答の履歴をAIが学習し、最適な復習タイミングを調整。
5. 継続するほど知識の定着度が可視化され、**“忘れない自分”を体感できる。**

→ 学習者は、**毎日5分で効率的な復習ができ、知識が自然と積み上がる**ようになります。

## サービスの差別化ポイント・推しポイント

| **項目** | **内容** |
| --- | --- |
| **① AI × 忘却曲線** | 復習タイミングをAIが自動最適化。従来の単なるスケジュール管理ではなく、**動的な出題設計**を実現。 |

## 機能候補

### MVPリリース時（Phase 1）

- アカウント登録・ログイン（**Drupal標準機能**）
- 復習したいトピック登録機能（`source_url`登録含む）
- **AIクイズ自動生成機能**（Cloud Functions経由）
- 本日の復習セッション（記述式・コード修正問題を優先）
- 正誤履歴と**回答時間**の記録
- **AIによる回答コードの簡易品質評価**
- 忘却曲線に基づく次回出題日計算
- マイダッシュボード（定着度、進捗、**弱点コンセプトのヒートマップ表示**）
- 設定ページ、法務・サポートページ

### 本リリース以降（Phase 2）

- バッジ機能（連続記録・成長メダル）
- **GitHub/Notion連携機能**（ソースコードや学習メモを自動でトピック化）
- 詳細な忘却曲線のグラフ・分析レポート
- 通知リマインダー機能
- 有料プラン（Pro）の課金機能実装
- **チーム学習・法人向け管理ダッシュボード**（Drupalの権限管理を活用）
